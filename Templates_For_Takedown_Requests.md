## üíæ EVIDENCE COLLECTION BEST PRACTICES

**For Each Instance, Capture**:

1. **Screenshot Requirements**:
   - Full page view
   - URL visible in browser
   - Date and time visible
   - Your computer clock (if possible)
   - Account profile picture and details
   - Follower count, engagement metrics

2. **Archive**:
   - Use archive.org (Wayback Machine)
   - Use archive.today
   - Use built-in PDF save in browser
   - Save full HTML if possible

3. **Technical Data**:
   - WHOIS data for websites
   - IP address (if accessible)
   - Platform account creation date
   - Previous posts from the account
   - Email addresses associated
   - Phone numbers (if visible)

4. **Documentation**:
   - Create spreadsheet tracking all instances
   - Organize by platform, date, severity
   - Track status of each removal request
   - Record all communication with platforms
   - Save all response emails/tickets

5. **Storage**:
   - Cloud storage (Google Drive, Dropbox) with password
   - Local backup on external drive
   - Shared access for legal team
   - Organized folder structure
   - Regular backups
